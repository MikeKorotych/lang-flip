● 01 · Vision

Что мы строим и почему это должно ощущаться магически

LangFlip снимает самый раздражающий налог с людей, которые весь день живут между английским, украинским и русским: неправильная раскладка, мелкие ошибки, перевод выделенного текста, текст с картинки и быстрые локальные AI-исправления.

Продуктовая формула простая: исправлять там, где уверенность высокая, просить явное действие там, где риск выше, и всегда оставаться локальным по умолчанию. Авто-флип работает на правилах и словарях, AI включается только когда пользователь настроил Ollama, OpenAI-compatible endpoint или Apple Foundation Models.

● 02 · System Architecture

Компоненты, события и границы ответственности

Архитектура держится на одном главном loop: EventTap видит явные жесты или завершение слова, принимает решение через локальные политики, затем аккуратно пишет обратно в активное приложение синтетическими клавишами.

Правило карты: EventTap остается тонким диспетчером. Решения про риск, язык, AI readiness и пользовательские исключения живут в отдельных компонентах.

  3. 01

    EventTap игнорирует свои synthetic events, проверяет enabled state, распознает Shift gestures и word boundaries.

  4. 02

    Авто-исправление молчит в рискованных местах, пропускает пароли и уважает выученные исключения.

● 03 · Product Surface

Поведение, которое пользователь должен запомнить телом

Single Shift
AI cleanup
Исправляет выделенный текст или последнюю фразу без выделения, если AI provider готов.
Double Shift
Primary layout flip
Выделение или последний token. English по умолчанию уходит в Ukrainian, non-English в English.
Triple Shift
Secondary layout flip
Тот же ручной путь, но с secondary language. Сейчас дефолтно Russian.
Shift + Space
Translate to current layout
Переводит выделение в язык активной раскладки и съедает rogue space.
Shift + Command + S
Copy text from screenshot
Работает только с vision-capable Ollama model, чтобы не красть системные shortcuts зря.
Both Shifts
Pause / resume
Моментальный режим тишины для mixed-language текста или чувствительных контекстов.

● 07 · Decisions

Архитектурные решения которые не хочется забыть

Lifecycle
Defer EventTap until onboarding grants permissions
Иначе macOS системный prompt появляется поверх нашего окна и ломает первый опыт.
Development
Never test the raw build copy as the daily path
TCC permissions привязаны к установленному signed app bundle, поэтому make dev копирует в /Applications.
Manual intent
Manual double/triple Shift does not require dictionary confidence
Когда пользователь явно просит flip, словарь не должен блокировать rare names, slang, commands or mixed tokens.
Privacy
Cloud AI is opt-in, local Ollama is the recommended path
Текст и screenshots остаются локальными, пока пользователь сам не внесет API key и provider.
Docs practice
Keep docs/plan.html as the visual operating manual
README отвечает на "что это", ROADMAP на "куда идем", CHANGELOG на "что изменилось", plan.html на "как все связано".

Gesture starts explicit intent

AI work starts from a user gesture: selected text, last sentence, screenshot region, or future push-to-talk.

    Recommended local model
    Qwen 3.5 2B
    Fast enough for short grammar fixes. 4B remains a heavier quality option for 16 GB+ Macs.
    Dictation baseline
    Whisper large-v3-turbo
    Good multilingual baseline for push-to-talk MVP, likely out-of-process first.
    TTS path
    System · Cloud · OmniVoice
    System voices are instant; OpenRouter cloud TTS gives better BYOK voices; OmniVoice stays local and heavier.
    Cloud TTS default
    GPT-4o Mini TTS via OpenRouter
    Use openai/gpt-4o-mini-tts-2025-12-15 for cost/compatibility, try Gemini 3.1 Flash TTS for richer multilingual speech, or Kokoro for cheap lightweight playback.
    Cloud STT default
    NVIDIA Parakeet via OpenRouter
    Dictation can now use local Whisper or nvidia/parakeet-tdt-0.6b-v3 through Cloud STT. Qwen3 ASR Flash is the noisy/mixed-language fallback.
    Cloud OCR default
    Gemini 3.1 Flash Lite via OpenRouter
    Screenshot text capture can stay local through Ollama or use a BYOK vision model. Default cloud OCR is google/gemini-3.1-flash-lite; Qwen, Gemma, and Perceptron are low-cost comparison options.
    Cloud text-fix default
    Gemini 3.1 Flash Lite via OpenRouter
    Single Shift and Fix Selected Text can use a curated cloud proofreading model. Default is google/gemini-3.1-flash-lite; GPT-5 Nano, DeepSeek V4 Flash, Qwen 3.6 Flash, GPT-5 Mini, and Mistral Medium are comparison options.
